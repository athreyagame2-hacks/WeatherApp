# ToDo CLI (Java)

A minimal command-line To-Do application in Java.  
Stores tasks in `tasks.txt` in the project folder.

## Features
- Add tasks
- List tasks with indices
- Mark tasks as done
- Remove tasks
- Persistent across runs (saved to `tasks.txt`)

## Run locally
1. Save `ToDoApp.java` to a folder.
2. Open terminal in that folder.
3. Compile:
   ```bash
   javac ToDoApp.java
